HOW TO START
------------
1. [Install NativeScript](https://docs.nativescript.org/start/quick-setup)
2. `git clone https://github.com/unlight/nativescript-typescript-template.git`
3. `tns install`
4. `sigh -w`
5. `tns livesync android --watch`

UNIT TESTS
----------
```
tns test android
```
